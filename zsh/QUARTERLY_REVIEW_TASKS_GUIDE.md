# Quarterly Review Tasks Guide

## üéØ Overview

A quarterly review is a strategic deep-dive into your entire GTD system. It's your time to assess progress, make major adjustments, and plan for the next quarter.

## ‚è±Ô∏è Time Commitment

**Recommended time:** 3-4 hours
**Best day:** Last weekend of the quarter or first weekend of new quarter
**Frequency:** Once per quarter (every 3 months)

---

## üìã Recommended Quarterly Review Tasks

### Phase 1: Comprehensive System Audit (60-90 minutes)

#### 1. **Complete System Cleanup**
- Archive all completed projects
- Archive all completed tasks
- Clean inbox completely
- Review and clean Waiting For
- Review and clean Someday/Maybe
- Archive old reference material

#### 2. **Project Portfolio Review**
- Review ALL projects (active, on-hold, completed)
- Assess project health
- Identify projects to pause or cancel
- Identify projects to activate
- Review project alignment with goals

#### 3. **Area Portfolio Review**
- Review ALL areas comprehensively
- Archive inactive areas
- Merge overlapping areas
- Refine area definitions
- Review area standards
- Assess area balance

#### 4. **Task Inventory**
- Review all tasks
- Delete obsolete tasks
- Update task priorities
- Reorganize tasks
- Identify task patterns

---

### Phase 2: Strategic Assessment (60-90 minutes)

#### 5. **Quarterly Goals Review**
- Review goals set at start of quarter
- Assess progress on each goal
- Celebrate achievements
- Identify goals that need adjustment
- Set goals for next quarter

#### 6. **Habit System Review**
- Review all habits
- Assess habit consistency
- Identify habits to add
- Identify habits to remove
- Review habit effectiveness
- Plan habit improvements

#### 7. **Financial Review**
- Complete quarterly financial review
- Review investment portfolio
- Review debt strategy
- Review insurance
- Review financial goals
- Plan financial improvements

#### 8. **Career and Professional Development**
- Review career goals
- Assess professional progress
- Review skill development
- Plan learning for next quarter
- Review networking activities

#### 9. **Health and Wellness Review**
- Review health goals
- Assess workout consistency
- Review nutrition patterns
- Review medication compliance
- Plan health improvements

#### 10. **Relationships Review**
- Review relationship area
- Assess relationship quality
- Identify relationships to nurture
- Plan relationship activities
- Review relationship goals

---

### Phase 3: Pattern Analysis (45-60 minutes)

#### 11. **Review All Daily Logs**
- Review entire quarter's daily logs
- Identify major patterns
- Extract key insights
- Note recurring themes
- Track energy patterns
- Track productivity patterns

#### 12. **Review All Weekly Reviews**
- Review all weekly reviews from quarter
- Identify weekly patterns
- Note recurring issues
- Celebrate weekly wins
- Extract quarterly insights

#### 13. **Review All Monthly Reviews**
- Review all monthly reviews from quarter
- Identify monthly patterns
- Track progress over time
- Note trends
- Extract strategic insights

#### 14. **Second Brain Comprehensive Review**
- Review entire Second Brain
- Update all MOCs
- Archive old notes
- Create new connections
- Express content from notes
- Review note quality
- Plan knowledge improvements

#### 15. **Identify Major Patterns**
- Productivity patterns
- Energy patterns
- Time usage patterns
- Spending patterns
- Habit patterns
- Challenge patterns
- Success patterns

---

### Phase 4: Strategic Planning (60-90 minutes)

#### 16. **Set Quarterly Goals**
- Review long-term goals
- Set 3-5 major quarterly goals
- Break down into monthly goals
- Create action plans
- Set milestones

#### 17. **Plan Major Projects**
- Identify key projects for quarter
- Set project milestones
- Plan project timelines
- Allocate resources
- Set project priorities

#### 18. **Review and Adjust Systems**
- Assess GTD system effectiveness
- Identify system improvements
- Adjust workflows
- Update processes
- Optimize tools
- Plan system enhancements

#### 19. **Review Calendar and Commitments**
- Review past quarter's calendar
- Review upcoming quarter's calendar
- Identify busy periods
- Plan for important events
- Block time for priorities
- Review commitments

#### 20. **Life Balance Assessment**
- Review all areas of life
- Assess balance
- Identify areas needing attention
- Plan for better balance
- Set balance goals

---
