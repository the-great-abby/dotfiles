# Monthly Review Tasks Guide

## üéØ Overview

A monthly review is a deeper, more strategic review than the weekly review. It's your time to step back, assess progress, make adjustments, and plan for the month ahead.

## ‚è±Ô∏è Time Commitment

**Recommended time:** 2-3 hours
**Best day:** Last weekend of the month or first weekend of new month
**Frequency:** Once per month

---

## üìã Recommended Monthly Review Tasks

### Phase 1: System Cleanup (30-45 minutes)

#### 1. **Archive Completed Projects**
- **Task:** Review and archive all completed projects
- **Why:** Keep your system clean and focused
- **Actions:**
  - Review all projects
  - Archive completed ones
  - Update project status
  - Move to archive folder

**How:**
```bash
gtd-project list
# Review each project
gtd-project archive "project-name"
```

#### 2. **Archive Completed Tasks**
- **Task:** Clean up completed tasks
- **Why:** Remove clutter from active lists
- **Actions:**
  - Review all tasks
  - Archive completed ones
  - Delete obsolete tasks
  - Update task status

**How:**
```bash
gtd-task list --status=done
# Archive completed tasks
```

#### 3. **Review and Clean Inbox**
- **Task:** Ensure inbox is truly empty
- **Why:** Start fresh each month
- **Actions:**
  - Process any remaining items
  - Delete obsolete items
  - Archive reference items
  - Ensure zero items

**How:**
```bash
gtd-process --all
```

#### 4. **Review Waiting For List**
- **Task:** Clean up waiting items
- **Why:** Follow up on overdue items
- **Actions:**
  - Review all waiting items
  - Follow up on overdue items
  - Archive resolved items
  - Update status

#### 5. **Review Someday/Maybe List**
- **Task:** Clean up future possibilities
- **Why:** Keep list relevant and inspiring
- **Actions:**
  - Remove items no longer interesting
  - Archive items that will never happen
  - Activate items ready to become projects
  - Add new ideas

---

### Phase 2: Strategic Assessment (45-60 minutes)

#### 6. **Comprehensive Project Review**
- **Task:** Review all projects strategically
- **Why:** Ensure projects align with goals
- **For each project:**
  - Is it still relevant?
  - Is it making progress?
  - Should it be on hold?
  - Does it need more resources?
  - What's the next milestone?

**Questions:**
- What projects are stuck?
- What projects need more attention?
- What projects should be paused?
- What projects should be activated?

#### 7. **Comprehensive Area Review**
- **Task:** Review all areas of responsibility
- **Why:** Ensure all life areas get attention
- **For each area:**
  - Is it still relevant?
  - Are standards being met?
  - What needs attention?
  - Should it be archived or merged?
  - What new projects are needed?

**How:**
```bash
gtd-area review-all
```

**Questions:**
- What areas are thriving?
- What areas need attention?
- What areas should be archived?
- What new areas should be added?

#### 8. **Review Monthly Goals**
- **Task:** Assess progress on monthly goals
- **Why:** Track progress and adjust
- **Actions:**
  - Review goals set at start of month
  - Assess progress on each goal
  - Celebrate achievements
  - Identify goals that need adjustment
  - Set goals for next month

**Questions:**
- What goals did I achieve?
- What goals need more work?
- What goals should be adjusted?
- What new goals should I set?

#### 9. **Review Habits and Routines**
- **Task:** Assess habit consistency
- **Why:** Maintain and improve routines
- **Actions:**
  - Review habit dashboard
  - Check streaks and consistency
  - Identify habits that need attention
  - Celebrate successful habits
  - Adjust or add habits

**How:**
```bash
gtd-habit dashboard
gtd-habit list
```

**Questions:**
- What habits are consistent?
- What habits need improvement?
- What habits should be added?
- What habits should be removed?

#### 10. **Review Financial Status**
- **Task:** Comprehensive monthly financial review
- **Why:** Track financial health
- **Actions:**
  - Complete monthly budget review
  - Review all accounts
  - Track net worth
  - Review expenses and spending
  - Review savings goals
  - Review investment performance

**How:**
```bash
make gtd-wizard ‚Üí 6) Review ‚Üí 3) Financial review ‚Üí 2) Monthly
```

### Phase 3: Pattern Recognition (30-45 minutes)

#### 11. **Review Daily Logs**
- **Task:** Review entire month's daily logs
- **Why:** Identify patterns and insights
- **Actions:**
  - Read through all daily logs
  - Identify patterns
  - Note insights and learnings
  - Extract key accomplishments
  - Identify recurring themes

**What to look for:**
- Accomplishments
- Patterns and trends
- Insights and learnings
- Recurring challenges
- Energy patterns
- Productivity patterns

#### 12. **Review Weekly Reviews**
- **Task:** Review all weekly reviews from the month
- **Why:** See weekly patterns and progress
- **Actions:**
  - Review all weekly review files
  - Identify weekly patterns
  - Note recurring issues
  - Celebrate weekly wins
  - Extract insights

**Questions:**
- What patterns do I see?
- What issues keep coming up?
- What's working well?
- What needs improvement?

#### 13. **Review Second Brain**
- **Task:** Comprehensive Second Brain review
- **Why:** Maintain knowledge base
- **Actions:**
  - Review notes that need refinement
  - Update MOCs
  - Archive old notes
  - Create connections
  - Express content from notes
  - Review note quality

**How:**
```bash
gtd-brain list
gtd-brain-moc list
gtd-brain-discover
gtd-brain-express
gtd-brain-metrics
```

#### 14. **Identify Patterns and Trends**
- **Task:** Look for patterns across all reviews
- **Why:** Make strategic adjustments
- **What to identify:**
  - Productivity patterns
  - Energy patterns
  - Time usage patterns
  - Spending patterns
  - Habit patterns
  - Challenge patterns

**Questions:**
- What patterns do I see?
- What trends are emerging?
- What needs to change?
- What should I continue?

---

### Phase 4: Planning Ahead (30-45 minutes)

#### 15. **Set Monthly Goals**
- **Task:** Set goals for next month
- **Why:** Focus and direction
- **Actions:**
  - Review long-term goals
  - Set 3-5 monthly goals
  - Break down into weekly goals
  - Create action plans
  - Set milestones

**Goal categories:**
- Personal development
- Health & wellness
- Work & career
- Relationships
- Financial
- Creative

#### 16. **Plan Major Projects**
- **Task:** Plan major projects for next month
- **Why:** Ensure progress on important work
- **Actions:**
  - Identify key projects
  - Set project milestones
  - Plan project tasks
  - Allocate resources
  - Set deadlines

#### 17. **Review Calendar**
- **Task:** Review and plan calendar
- **Why:** Manage commitments and time
- **Actions:**
  - Review past month's calendar
  - Review upcoming month's calendar
  - Identify busy periods
  - Plan for important events
  - Block time for priorities

#### 18. **Review and Adjust Systems**
- **Task:** Review GTD and Second Brain systems
- **Why:** Keep systems effective
- **Actions:**
  - Review system usage
  - Identify improvements
  - Adjust workflows
  - Update processes
  - Optimize tools

**Questions:**
- Is my system working well?
- What needs improvement?
- What processes should I change?
- What tools should I adjust?

#### 19. **Review Relationships and Social**
- **Task:** Review relationships and social connections
- **Why:** Maintain important connections
- **Actions:**
  - Review relationship area
  - Identify people to connect with
  - Plan social activities
  - Review relationship goals
  - Schedule time for relationships

#### 20. **Review Health and Wellness**
- **Task:** Comprehensive health review
- **Why:** Maintain physical and mental health
- **Actions:**
  - Review health area
  - Assess workout consistency
  - Review nutrition
  - Check medication compliance
  - Review health goals
  - Plan health improvements
