## ðŸ’¡ Quick Commands Reference

After learning, you'll know these commands:

### Basic Operations
```bash
gtd-brain create "Note Title" Resources    # Create a note
gtd-brain-sync                              # Sync GTD with Second Brain
gtd-brain list                              # List all notes
gtd-brain search "term"                     # Search notes
```

### Organization
```bash
gtd-brain-moc create "Topic"               # Create MOC
gtd-brain-template create "Template" "Name" # Use template
gtd-brain move <note> Projects              # Move note
```

### Knowledge Building
```bash
gtd-brain-distill <note>                    # Progressive summarization
gtd-brain-evergreen mark <note>             # Mark as evergreen
gtd-brain-connect create <note1> <note2>    # Create connection
```

### Content Creation
```bash
gtd-brain-express create "Title" "notes"   # Create content
gtd-brain-packet create <note> <name>        # Create packet
```

### Quality & Maintenance
```bash
gtd-brain-metrics dashboard                 # Quality dashboard
gtd-brain archive-review                    # Review for archiving
```
